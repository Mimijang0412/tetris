# Tetris

A classic Tetris game built with vanilla JavaScript and HTML5 Canvas. No frameworks, no dependencies — just open and play.

## Features

- SRS (Super Rotation System) wall kicks
- Ghost piece drop preview
- Next piece preview
- Line-clear animations
- Progressive difficulty with increasing levels
- Score tracking with soft/hard drop bonuses

## How to Play

Open `index.html` in your browser and press **Enter** to start.

| Key | Action |
|-----|--------|
| ← → | Move left/right |
| ↑ | Rotate |
| ↓ | Soft drop |
| Space | Hard drop |
| P | Pause |

## Scoring

| Action | Points |
|--------|--------|
| Soft drop | 1 per row |
| Hard drop | 2 per row |
| 1 line | 100 × level |
| 2 lines | 300 × level |
| 3 lines | 500 × level |
| 4 lines (Tetris) | 800 × level |
